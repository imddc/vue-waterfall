1. person's habit
2. create Vue projects faster and more conveniently

## Features

- file base routing
- components auto importing
- apis auto importing
- dev tools
- typescript
- deploy on vercel, zero-config

## Pre-packed

- unocss
- vue-router
- pinia
- vueuse
- vitest

## Usage

### development

run and visit localhost:9527

```bash
 pnpm install
 pnpm dev
```

### build

```bash
 pnpm build
```

And you will see the generated file in `dist` that ready to be served.
